Dynamic Book Reader

Unfortunately, I couldn't keep the code but I recorded the functionality below!

Responsive Design

To address accessablility issues with the previous book reader, I coded a new book reader with a responsive design. There are three different layouts: portrait with one page, portrait with two pages, and landscape with picture and caption on the left and paragraphs on the right.

Highlight Feature

Created a highlighting feature that displayed popups for word clicks and multiple word selections. Utilized a custom API service for persistent highlighting data.
